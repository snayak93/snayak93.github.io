This is my Portfolio Page!


Project 1: Beautifully Built Cars (BBC)

Our project is a game were you can customize your dream car. The setting is a car garage and the game asks you a sequence of questions that helps you desighn your car. Some of these questions include what type of body you want, what size wheels you want, and if you want a top hat or not.

Project 2: Tic Tac Toe

Our project is a tic tac toe game where users can enter their names to be displayed on a leaderboard. After the game ends the leaderboard displays and users can click again if they want to play the game again.

Project 3: Minion Runner

Our project is a minion runner game where the user has to move forward, backwards, and jump to traverse through the stages and help Bob find his banana.

Project 4: Phishy Fishtanks

For this project we investigated problems with a failing fishtank and solved the issues, strengthening the tank's cybersecurity and saving the fish.

Project 5: Rover

For this project we investigated the location of a lost rover by analyzing raw data obtained from sensors on the rover. We concluded the rover was likely in the rocky mountains.
