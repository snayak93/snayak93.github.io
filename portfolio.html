Project 1: Beautifully Built Cars (BBC)

Our project is a game were you can customize your dream car. The setting is a car garage and the game asks you a sequence of questions that helps you desighn your car. Some of these questions include what type of body you want, what size wheels you want, and if you want a top hat or not.\

Project 2: Tic Tac Toe

Our project is a tic tac toe game where users can enter their names to be displayed on a leaderboard. After the game ends the leaderboard displays and users can click again if they want to play the game again.
